# Barcode-Scanner

Description - 
-----------------------------------------

This appliction has been created to test the zxing library of google for barcode scanning support. This app scans both the barcode/
qrcode and shows the website (if any) in the webview of the app. If a text is present then it loads it in the text View and displays it
to the user.

Developed By -
--------------
Suvam Jain


Pre-requisites
--------------

- Android SDK v17

Getting Started
---------------

This sample uses the Gradle build system. To build this project, use the
"gradlew build" command or use "Import Project" in Android Studio.
